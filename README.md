# Snake: Infinity
🐍 Traditional game of Snake

## Deadline 
- DDL1: 7/10
- DDL2: next semester

## Requirements
在本项目中，你将独自实现一个贪吃蛇游戏，具体内容包括
1. 编程环境的搭建(5%)
2. 基本功能(35%)
    1. 蛇的显示和移动
    2. 墙和食物的显示与判定
3. 进阶功能(30%)
   1. 单机多人游戏(多条蛇，不同按键控制不同蛇)
   2. 具有特殊效果的食物(具体效果请发挥你的想象力)
   3. 初始地形以及随机产生的砖块
   4. 简单的AI蛇(策略自拟，如按照最短路径去吃最近的食物等，无需实现的非常复杂)
4. 其他功能(10%)
    除上述功能外，增加至少一项其他功能
5. 面向对象的设计、注释以及代码风格(20%)
